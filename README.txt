Pour utiliser le CMakeLists.txt :

Lancer dans le bash à la racine du dossier la commande suivante :
cmake -G"Unix Makefiles"

La commande précédente mets à jour le Makefile. Pour mettre à jour l'ex:
make

Si erreur : contacter votre binôme par voie orale , FB, ou via son adresse mail.
Ou tenter de comprendre par soi-même, c'est bien aussi.
